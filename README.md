# Réseau de Neurones Artificiels – Classification Binaire 2D

## Objectif
Construire un modèle de réseau de neurones artificiel (ANN) capable de classifier des points en deux catégories ("pos" et "neg") à partir de données bidimensionnelles.

## Données
- Fichier : artificial_generator.xlsx  
- 2000 exemples  
- Features : X1, X2  
- Cible : Y (pos / neg)

## Méthodologie
- Préparation et encodage des données
- Construction d’un modèle ANN avec TensorFlow / Keras
- Entraînement du modèle
- Évaluation des performances (accuracy, loss)
- Visualisation de la frontière de décision

## Outils
- Python  
- TensorFlow / Keras  
- Pandas  
- Scikit-learn  
- Matplotlib  

## Contenu du repository
- Notebook du projet : `Reseau_de_Neurones_Artificiels_Classification2D.ipynb`
- Rapport du projet : `Rapport_RNA_Classification_Binaire.pdf`

## Résultat
Le modèle obtient une bonne précision de classification et apprend efficacement la séparation entre les deux classes dans l’espace 2D.
